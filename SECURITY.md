# Security Policy

## Supported versions

Security fixes are provided for the latest published version of Furigana Keyboard.
Users should update to the latest version before reporting an issue that may already
have been fixed.

## Reporting a vulnerability

Please do not disclose suspected vulnerabilities in a public issue. Send a report
to `support@hanlu.app` with:

- the affected app version and platform;
- steps to reproduce the issue;
- the expected and actual behavior;
- any proof of concept, logs, or screenshots that do not contain sensitive input.

Do not include passwords, signing keys, private dictionary-update keys, or text
entered into third-party apps. The report will be acknowledged after it has been
reviewed, and disclosure timing will be coordinated when a fix is required.
