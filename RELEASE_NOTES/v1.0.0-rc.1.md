# Furigana Keyboard v1.0.0-rc.1

正式版候補の初回リリースです。

## 主な変更

- `downloads.hanlu.app`から直接配布版の更新を確認できる機能を追加
- 更新APKのSHA-256、パッケージ名、バージョン、署名証明書をインストール前に検証
- 設定画面とAbout画面の構成、表示、アクセシビリティを改善
- 手書き認識、ローマ字入力、かな漢字変換、文節変換の品質と安定性を改善
- Androidの依存関係とビルド環境を更新

## リリース候補版について

- 正式公開前の最終確認用ビルドです。重要な文章では変換結果を確定前に確認してください。
- 直接配布版の更新確認にはネットワーク接続が必要です。入力文字や手書きデータは更新確認に送信されません。

## 動作環境

- Android 7.0以降
- APKを直接インストールする場合、端末側で配布元からのインストール許可が必要になることがあります。
