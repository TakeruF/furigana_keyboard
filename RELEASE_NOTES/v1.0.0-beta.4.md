# Furigana Keyboard v1.0.0-beta.4

画面の安全領域とキーボード候補表示を改善したベータ版です。

## 主な変更

- Android 15で設定画面上部がステータスバーと重なる問題を修正
- オンボーディング画面をステータスバー、ノッチ、ナビゲーションバーの安全領域へ対応
- 端末側の「キーボードを持ち上げる」設定とジェスチャーナビゲーションの下端余白を反映
- 変換候補同士を細い縦線で区切り、候補と読み仮名を見分けやすく改善
- 先頭候補に表示していたピル状の背景を削除
- 候補間の余白を調整し、読み仮名同士が重ならない幅を維持
- 手書き入力と日本語ローマ字入力のスペースキーから文字表示を削除
- 候補バーを表示しない英字キーボード上部に小さな余白を追加

## ベータ版について

- 手書き認識、読み、変換候補の正確性は保証されません。重要な文章では確定前に内容を確認してください。
- 認識対象外の文字、固有名詞、辞書にない語では適切な候補が表示されない場合があります。
- 端末や入力先アプリによって、キーボードの表示や動作が異なる場合があります。

## 動作環境

- Android 7.0以降
- APKを直接インストールする場合、端末側で配布元からのインストール許可が必要になることがあります。
