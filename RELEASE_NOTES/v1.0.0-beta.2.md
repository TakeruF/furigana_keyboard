# Furigana Keyboard v1.0.0-beta.2

日本語入力と変換候補を中心に改善したベータ版です。

## 主な変更

- 日本語タイピングと手書き入力のうち、最後に使用したキーボードを次回も表示するように改善
- カタカナ入力時にカタカナの変換候補を表示するように改善
- 動詞・形容詞などの活用形を変換候補へ追加
- 「考えて」「使えて」「使え」など、入力した活用形そのものを候補から選びやすく改善
- 英字キーボードの句読点を `,` と `.` に変更
- 日本語表示の空白キーを「スペース」に変更
- 日本語のローマ字入力と削除処理を調整

## ベータ版について

- 手書き認識、読み、変換候補の正確性は保証されません。重要な文章では確定前に内容を確認してください。
- 認識対象外の文字、固有名詞、辞書にない語では適切な候補が表示されない場合があります。
- 端末や入力先アプリによって、キーボードの表示や動作が異なる場合があります。

## 動作環境

- Android 7.0以降
- APKを直接インストールする場合、端末側で配布元からのインストール許可が必要になることがあります。
