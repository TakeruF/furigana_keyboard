# Furigana Keyboard v1.0.0-rc.2

正式版候補の第2版です。

## 主な変更

- 手書き入力中の削除キーで、最後に書いた文字を1文字ずつ消せるように改善
- 2文字を横並びで書いた際の文字分割と再認識を改善
- AndroidとiOSで手書き入力の操作を統一
- 高速なローマ字入力と手書き認識のテストを更新

## リリース候補版について

- 正式公開前の確認用ビルドです。重要な文章では変換結果を確定前に確認してください。
- 直接配布版の更新確認にはネットワーク接続が必要です。入力文字や手書きデータは更新確認に送信されません。

## 動作環境

- Android 7.0以降
- APKを直接インストールする場合、端末側で配布元からのインストール許可が必要になることがあります。
