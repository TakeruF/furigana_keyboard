# Furigana Keyboard v1.0.0-beta.3

キーボード表示とアプリ更新案内を改善したベータ版です。

## 主な変更

- 英字入力時に日本語向けの変換候補を表示しないようにし、英字キーボードの動作を簡素化
- Google Playで配信されている新しいバージョンを定期的に確認する機能を追加
- 新しいバージョンが利用可能な場合に、設定画面と通知から更新を確認できるように改善
- AboutページのAndroid APKリンクをv1.0.0-beta.3へ更新
- Aboutページのヘッダーを「Furigana Keyboard Beta」に変更
- APKダウンロード表示をAndroidアイコンに統一

## ベータ版について

- 手書き認識、読み、変換候補の正確性は保証されません。重要な文章では確定前に内容を確認してください。
- 認識対象外の文字、固有名詞、辞書にない語では適切な候補が表示されない場合があります。
- 端末や入力先アプリによって、キーボードの表示や動作が異なる場合があります。

## 動作環境

- Android 7.0以降
- APKを直接インストールする場合、端末側で配布元からのインストール許可が必要になることがあります。
