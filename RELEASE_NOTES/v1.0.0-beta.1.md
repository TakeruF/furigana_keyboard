# Furigana Keyboard v1.0.0-beta.1

Furigana Keyboardの最初の公開ベータ版です。

手書きした日本語を端末上で認識し、読み付きの候補から入力できます。文字の形は分かるけれど
読み方に迷うときや、通常の変換では目的の漢字を見つけにくいときに利用できます。

## 主な機能

- ひらがな・カタカナ・JIS X 0208漢字のオフライン手書き認識
- 候補へのふりがな表示
- 手書き欄へ1文字、または2文字を横並びで書く入力
- JMdictを利用した単語候補と読み検索
- 日本語ローマ字入力と、文脈に沿った変換候補
- QWERTY、数字・記号、手書き入力の切り替え
- 日本語、簡体字中国語、英語、韓国語の画面表示

## Furigana Plus

Furigana Plusを有効にすると、Google ML Kitの端末内日本語認識モデルを利用できます。認識に失敗
した場合は、アプリ内蔵のオフライン認識へ自動的に切り替わります。

Furigana Plusは初期状態で有効です。初回利用時に約20 MBのモデルをダウンロードするため、通信が
必要です。地域や端末環境によってダウンロードまたは認識が安定しない場合は、設定の「手書き入力」
からオフにできます。オフにしても基本の手書きキーボードは利用できます。

## ベータ版について

- 手書き認識、読み、変換候補の正確性は保証されません。重要な文章では確定前に内容を確認して
  ください。
- 認識対象外の文字、固有名詞、辞書にない語では適切な候補が表示されない場合があります。
- 端末や入力先アプリによって、キーボードの表示や動作が異なる場合があります。
- 問題を見つけた場合は、端末機種、Androidバージョン、再現手順を添えてご連絡ください。

## 動作環境

- Android 7.0以降
- APKを直接インストールする場合、端末側で配布元からのインストール許可が必要になることが
  あります。

iOS版は現在MVP開発段階であり、このAndroid APKには含まれません。

## プライバシー

入力文字、手書きストローク、認識結果をアプリの運営者へ送信・保存する機能はありません。
基本の認識と辞書検索は端末内で行われます。任意機能であるFurigana Plusのモデル配信にはGoogleの
サービスを使用します。詳しくはアプリ内のプライバシーポリシーをご確認ください。
