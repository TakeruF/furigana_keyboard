# Furigana Keyboard v1.0.0-beta.5

オフラインのかな漢字変換と更新機能を強化したベータ版です。

## 主な変更

- 日本語ローマ字入力に、端末内で動作するかな漢字変換を追加
- 単語のつながりを考慮した変換候補を生成し、長い入力の候補精度を改善
- 読み辞書を拡充し、単語候補と読み仮名の収録範囲を拡大
- 読み・変換辞書を安全に更新できる仕組みを追加
- Android直接配布版に、署名とハッシュを検証するアプリ更新機能を追加
- 高速入力時の取りこぼしを抑え、QWERTYキーボードのキー配置と操作性を調整
- Android、iOS、Aboutページのアプリアイコンを刷新
- Aboutページに繁体字中国語を追加し、中国語の言語判定と法的文書を改善

## ベータ版について

- 手書き認識、読み、かな漢字変換の正確性は保証されません。重要な文章では確定前に内容を確認してください。
- 認識対象外の文字、固有名詞、辞書にない語では適切な候補が表示されない場合があります。
- 端末や入力先アプリによって、キーボードの表示や動作が異なる場合があります。

## 動作環境

- Android 7.0以降
- APKを直接インストールする場合、端末側で配布元からのインストール許可が必要になることがあります。
