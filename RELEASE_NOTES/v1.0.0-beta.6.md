# Furigana Keyboard v1.0.0-beta.6

日本語ローマ字入力に、文節単位の変換と確定を追加したベータ版です。

## 主な変更

- 候補欄の「文節」から、文章を文節ごとに変換できる機能を追加
- 変換中の文節を背景色と下線で表示し、現在の変換対象を分かりやすく改善
- 候補欄の左右操作で、変換対象の文節を短くしたり次の文節まで広げたりできるように変更
- 候補を選択したとき、選択中の文節だけを確定するように対応
- 確定していない後続部分を削除せず、次の変換対象として保持
- Backspaceで、確定済みの文字より未確定の後続部分を優先して編集するように改善
- 変換エンジンのsegment情報を文節境界に利用し、情報がない場合は助詞などを考慮して分割
- 従来の全文変換、候補選択、スペースキー、「変換」キーの操作を維持

## 文節変換の使い方

1. 日本語ローマ字入力で文章を入力します。
2. 候補欄の「文節」を選択します。
3. 背景色と下線で示された文節の候補を選びます。
4. 選択した文節だけが確定され、残りの文章が次の変換対象として残ります。
5. 必要に応じて候補欄の「←」「→」で変換範囲を調整します。

例えば「わたしもそうしたい」は、「私も」を確定した後も「そうしたい」が未確定のまま残り、続けて変換できます。

## ベータ版について

- 文節境界やかな漢字変換の正確性は保証されません。重要な文章では確定前に内容を確認してください。
- 固有名詞、辞書にない語、口語表現では適切な文節や候補が表示されない場合があります。
- 端末や入力先アプリによって、変換中の背景色や下線の表示が異なる場合があります。

## 動作環境

- Android 7.0以降
- APKを直接インストールする場合、端末側で配布元からのインストール許可が必要になることがあります。
